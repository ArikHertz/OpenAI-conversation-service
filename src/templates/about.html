System Information

This system demonstrates a ChatGPT-style web application built using Flask, MySQL and OpenAI. It uses server-side rendering with Jinja2 templates, stores every conversation in a MySQL database, and sends the full message history to OpenAI so the assistant can maintain context within a session.

  • Backend: Python Flask
  • Database: MySQL with SQLAlchemy
  • AI: OpenAI Chat Completions API
  • Frontend: HTML, CSS, JavaScript, Bootstrap 5
